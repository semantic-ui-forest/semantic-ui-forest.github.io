Semantic-UI-Forest

Collection of design, themes and templates for Semantic-UI.

Features

Beautiful

Born to be beautiful. Elaborate cross-platform web fonts solution for both Chinese and English.

  • Semantic-UI, the foundation.
  • fonts.css, for Chinese.
  • Linux Libertine, for English.

Versatile

Support multiple documentation markup language via pandoc.

  • Markdown
  • Org
  • reStructuredText
  • Textile

Flexible

Manage all kinds of resource files in a uniform and flexible way.

  • Documents
  • HTML & CSS & JavaScript
  • Images & Videos
  • Fonts

Batteries Included

Builtin generators for multiple components of a modern website.

  • robots.txt
  • humans.txt
  • Atom Feed
  • Sitemap

Dynamic

Generate dynamic contents from static contents, sounds like a magic?

  • Tags
  • Categories
  • Paginator
  • List of Documents

Responsive

Born to be Responsive. Write once, fit everywhere. Perfect experience on all devices.

  • Phone
  • Tablet
  • Desktop

Scienticfic!

Excellent math formula and syntax highlighting support for math and technical writings.

  • MathJax
  • Optional PDF genaration
  • Source code syntax highlighting

Integrations

Easy to integrate third-party services and bring lives and noises(I guess so) to your site.

  • Google Analytics for statistics
  • Disqus for commenting

Internationalization

One click to switch languages for the whole site, or just a single post or note.

  • Built-in localization for common UI components.

Guidance

Best practice guidance for a smooth and happy work flow.

  • Guard
  • Browsersync
  • Github Pages
  • Travis-CI

Compatible

Nanoc-Plate is just a bundle of nanoc. No wonder that we have all features of nanoc.

  • Free-form metadata
  • Programmable filters and helpers
  • Various templating languages

Customizable

Want to do some more in-depth customization? Not that hard, we've made great choices already.

  • Slim, a better HTML.
  • Sass, a better CSS.
  • Vanilla JavaScript? Plus jQuery, just enough here.
